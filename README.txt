PROJECT - SENTIMENT
First Attempt of a self-learning Intelligence
GOALS: To be able to comprehend human emotion through text and to provide artificial emotion

Method 21_11_01------------------------------------------------------------------------------
Program is able to intake information and produce a somewhat premature guess (Due to lack of
data provided) on whether it was being insulted or not.
--------------------------------------------------------------------------------------------

Method 22_05_27------------------------------------------------------------------------------
Program is able to take the information on whether it is being insulted or complimented and
transfer it into a scale (0 - 10) of preset emotions and provide an artificial emotion back
to the user (note: emotions reset when program is started)
--------------------------------------------------------------------------------------------
